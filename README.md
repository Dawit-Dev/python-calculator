# Python Calculator

A simple command-line calculator built with Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation
- Division by zero protection
- Continuous calculator loop
- Perform multiple calculations without restarting the program

## Technologies

- Python 3
- Git
- GitHub
- VS Code

## How to Run

```bash
python3 calculator.py
```
